3. Operators

Highlight Strict Equality (===) vs Loose Equality (==). 
let a = 10, b = 5;

// Arithmetic
console.log("Sum:", a + b); // 15
console.log("Is a > b?", a > b); // true

// The "Tricky" Part
console.log(a == "10");  // true (Values match)
console.log(a === "10"); // false (Types don't match: Number vs String)

6. Functions

Compare function keyword vs Modern Arrow Functions (=>). 
// 1. Traditional Function
function greet(name) {
  return "Hello " + name;
}
console.log(greet("Bob"));

// 2. Modern Arrow Function
const square = (x) => x * x;

console.log("Square of 4 is:", square(4)); // 16

8 DOM Manipulation

Interacting with the webpage: selecting elements, changing content, and handling events. 
// 1. Select Elements
const title = document.querySelector('h1');
const btn = document.getElementById('myBtn');

// 2. Change Content & Style
title.innerText = "Hello from JS!";
title.style.color = "cyan";

// 3. Event Listeners
document.addEventListener('click', () => {
    console.log("Document clicked!");
});

// Example: Create a button dynamically
const newBtn = document.createElement('button');
newBtn.innerText = "Click Me";
newBtn.className = "bg-blue-500 text-white p-2 rounded mt-2";
document.body.appendChild(newBtn);

newBtn.addEventListener('click', (e) => {
    e.stopPropagation(); // Stop bubbling
    alert("Button Clicked!");
});
