üìö Part 1: How to Learn Tailwind

Understand the system first!

The Secret

You DON'T need to memorize thousands of class names! Tailwind follows predictable patterns. Once you understand the pattern, you can guess most class names correctly!

üé® The Pattern Formula

BASIC PATTERN

[property]-[value]
bg-blue-500

background + blue + shade 500

text-xl

text size + extra large

p-4

padding + size 4

m-8

margin + size 8

WITH DIRECTION

[property][direction]-[value]
pt-4

padding-top + 4

ml-2

margin-left + 2

px-6

padding horizontal (left+right) + 6

py-3

padding vertical (top+bottom) + 3

WITH STATES

[state]:[property]-[value]
hover:bg-blue-600

on hover ‚Üí background blue 600

focus:ring-2

on focus ‚Üí ring width 2

md:grid-cols-3

on medium screens ‚Üí 3 columns

lg:text-4xl

on large screens ‚Üí text size 4xl

üé® Part 2: Live Interactive Examples

Where do these class names come from?

Tailwind CSS provides pre-defined utility classes. Instead of writing CSS, you just combine these classes!

1. Colors üé®

bg-blue-500

Background Blue

bg- = background, blue = color, 500 = shade (100=lightest, 900=darkest)

text-red-600

Red Text Color

text- = text color

Available colors: red, orange, yellow, green, blue, purple, pink, gray, etc.

2. Spacing (Padding & Margin) üìè

p-4

Padding on all sides (1rem = 16px)

p- = padding, 4 = 1rem (16px)

px-6 py-2

px = horizontal (left+right), py = vertical (top+bottom)

px- = padding horizontal (left & right)
py- = padding vertical (top & bottom)

m-4

Margin on all sides

m- = margin

Spacing Scale:
0 = 0px, 1 = 0.25rem (4px), 2 = 0.5rem (8px), 4 = 1rem (16px), 6 = 1.5rem (24px), 8 = 2rem (32px), etc.

3. Text Styling ‚úçÔ∏è

text-xl font-bold

Extra Large Bold Text

text-xl = font size (xs, sm, base, lg, xl, 2xl, 3xl, 4xl...)
font-bold = font weight

text-center italic

Centered Italic Text

text-center = text alignment (also: text-left, text-right)

underline uppercase

Underlined Uppercase

4. Flexbox Layout üì¶

flex items-center justify-between

Left
Right

flex = display flex
items-center = vertical alignment (center)
justify-between = horizontal spacing (space between items)

gap-4 = space between flex items

grid-cols-3 = 3 columns
gap-4 = space between grid items

border-2 = 2px border width

grid-cols-1 = 1 column on mobile
md:grid-cols-2 = 2 columns on tablets (768px+)

Breakpoints:

10. Gradients üåà

Left to Right Gradient

bg-gradient-to-r = gradient direction (to right)
from-pink-500 = start color
to-purple-500 = end color

bg-gradient-to-br from-blue-500 via-purple-500 to-pink-500

3-Color Gradient

via-purple-500 = middle color (optional)
