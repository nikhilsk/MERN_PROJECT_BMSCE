ü§î What is the Box Model?

In CSS, every element is treated as a rectangular box. The box model describes how the element's space is calculated, consisting of four parts:

  • Content - The actual content (text, images, etc.)
  • Padding - Space between content and border (transparent)
  • Border - The border around padding and content
  • Margin - Space outside the border (transparent)

üìê Total Element Size Formula
Total Width = Content Width + Padding Left + Padding Right + Border Left + Border Right + Margin Left + Margin Right

‚öôÔ∏è The box-sizing Property

The box-sizing property controls how the total width and height are calculated:

  • content-box (default) - Width/height applies to content only. Padding and border are added extra.
  • border-box (recommended) - Width/height includes content, padding, and border!

üß≠ Directional Properties

You can control each side of the box individually:

Padding

padding-top: 10px; padding-right: 20px; padding-bottom: 10px; padding-left: 20px; /* Shorthand: */ padding: 10px 20px;

Margin

margin-top: 15px; margin-right: auto; margin-bottom: 15px; margin-left: auto; /* Shorthand: */ margin: 15px auto;

Border

border-top: 2px solid red; border-right: 3px dashed blue; border-bottom: 2px solid red; border-left: 3px dashed blue;

üí° Pro Tips for Using the Box Model

  • Always use box-sizing: border-box - Makes your life easier! Sizes become predictable.
  • Use margin: 0 auto - Centers block elements horizontally.
  • Padding for internal spacing - Use padding to create space inside elements.
  • Margin for external spacing - Use margin to create space between elements.
  • Inspect with DevTools - Right-click ‚Üí Inspect ‚Üí See the box model visualization.
  • Negative margins are allowed - You can use negative values to overlap elements.
  • Margins can collapse - Vertical margins between elements can merge (collapse).
  • Percentage values - Padding/margin percentages are based on the parent's width.
